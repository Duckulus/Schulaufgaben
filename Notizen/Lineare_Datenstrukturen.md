# Lineare Datenstrukturen

## Allgemeines

### Beispiele
- Array
- Stack
- Queue
- Forward Linked List
- Doubly Linked List

### Aufbau
Lineare Anordnung von Daten

### Operationen
Bei dynamisch linearen Datentypen kann man
- Daten hinzufügen
- Daten löschen
- Daten verändern
- Daten lesen

### Organisation in Klassen
Linearer Aufbau
Dynamische Datenstrukturen haben meistens eine Instanz der `Node` Klasse für jedes Element.

### Implementierung

#### Queue
```java
import java.util.*;
Queue<Integer> queue = new LinkedList<Integer>();
```

#### Stack
```java
import java.util.Stack<E>;
Stack<Integer> myStack = new Stack<Integer>();
```
